# Group Aphasia Meeting App
This is a proof of concept application for supporting aphasia group meetings. 

## Participant Interface
The application has a participant side. From this side, participants may input their names, select the participant option and have access to an interface for composing text and one for sending quick template text.
Access a live version of the participant side [here](http://mmachiobiorah.com/group_aphasia/client/client.html). The code for this interface is in the "client" folder.

## Public-facing Interface
The public facing side of the interface assumes a large screen where participant messages may be displayed.
Access a live version of the public facing side [here](http://mmachiobiorah.com/group_aphasia/server/index.html).
The code for this interface is in the "server" folder.
